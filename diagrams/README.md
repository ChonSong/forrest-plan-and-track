# Diagrams

This directory contains visual representations of the mission, engine, and timeline.

## Contents

| File | Type | What It Shows |
|------|------|--------------|
| `engine-loop.mmd` | Mermaid | The propose → simulate → score → commit loop |
| `data-model.mmd` | Mermaid | 4-entity data model (Scenario, Run, Experiment, Finding) |
| `timeline.mmd` | Mermaid | 14-day Gantt-style timeline |
| `engine-loop.html` | HTML | Interactive engine loop visualization |
| `data-model.html` | HTML | Rich data model diagram |
| `timeline.html` | HTML | Rich timeline with phase details |

## How to View

- **Mermaid (`.mmd`):** Open in any Mermaid renderer (mermaid.live, VS Code Mermaid extension, or GitHub renders them natively)
- **HTML (`.html`):** Open in a browser for interactive/rich visuals
